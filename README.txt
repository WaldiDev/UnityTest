Steuerung:

Der Spieler kann mit den ASDW Tasten gesteuer werden.
Mit Space wird das In-Game Menü aufgerufen.

-----------
Info:

Das Spielfeld ist ein Irrgarten welcher mithilfe von Prim's Algorithmus generiert wird.
Im Hauptmenü kann ein Seed für die Generierung ausgewählt werden. Wird der Seed auf einen negativen Wert gesetzt
wird ein zufälliger Wert generiert.
